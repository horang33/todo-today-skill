# 📋 Todo Today - Claude 오늘의 할일 스킬

노션, 슬랙, 구글 캘린더에서 할일을 자동으로 수집해서 우선순위별로 정리해주는 Claude 스킬입니다.

## ✨ 주요 기능

- **멀티소스 수집**: 노션 태스크, 슬랙 멘션, 구글 캘린더 일정을 한 번에 확인
- **어제 할일 자동 추적**: 캘린더/노션 수정 이력 기반으로 완료 여부 자동 체크
- **우선순위 분류**: 긴급/중요/일반으로 자동 분류
- **미완료 이월**: 어제 못 한 일은 오늘 목록으로 자동 이월

## 🛠️ 사전 준비

### 1. Claude 연동 설정

Claude에서 아래 서비스들을 연결해야 합니다:

| 서비스 | 용도 | 필수 여부 |
|--------|------|----------|
| **Notion** | 태스크/할일 데이터베이스 조회 | 선택 |
| **Slack** | 멘션된 메시지 검색 | 선택 |
| **Google Calendar** | 오늘 일정 조회 | 선택 |

> 💡 최소 1개 이상 연결하면 사용 가능합니다. 연결 안 된 서비스는 자동으로 건너뜁니다.

### 2. 본인 정보 확인

스킬 설정에 필요한 정보들입니다:

| 항목 | 확인 방법 | 예시 |
|------|----------|------|
| **슬랙 유저 ID** | 슬랙 프로필 > 더보기 > Member ID 복사 | `U0ABC12345` |
| **슬랙 이름/닉네임** | 슬랙에서 멘션할 때 쓰는 이름 | `john`, `홍길동` |
| **담당 팀/프로젝트명** | 노션 검색 시 필터링할 키워드 (선택) | `마케팅팀`, `Project-X` |

## 📥 설치 방법

### 방법 1: Claude 스킬로 등록 (추천)

1. Claude 설정 > Features > Skills로 이동
2. "Add Skill" 클릭
3. `SKILL.md` 파일 내용을 붙여넣기
4. 아래 **커스터마이징** 섹션 참고해서 본인 정보로 수정
5. 저장

### 방법 2: 프롬프트로 직접 사용

스킬 등록 없이 대화에서 SKILL.md 내용을 붙여넣고 "이 방식으로 오늘 할일 정리해줘"라고 요청해도 됩니다.

## ⚙️ 커스터마이징

`SKILL.md` 파일에서 아래 부분을 본인 정보로 수정하세요:

### 필수 수정

```markdown
# Line 59 - 슬랙 유저 ID
변경 전: `<@YOUR_USER_ID> after:YYYY-MM-DD`
변경 후: `<@U0ABC12345> after:YYYY-MM-DD`  ← 본인 ID로 변경

# Line 60 - 슬랙 이름
변경 전: `your_name after:YYYY-MM-DD`
변경 후: `홍길동 after:YYYY-MM-DD`  ← 본인 이름으로 변경
```

### 선택 수정

노션에서 특정 팀/프로젝트 문서만 검색하고 싶다면 주의사항 섹션에 추가:

```markdown
# 주의사항 섹션
- 본인 담당 팀/프로젝트 문서만 필터링 (마케팅팀, Project-X 관련)
```

## 🎯 사용법

Claude에게 아래와 같이 말하면 스킬이 실행됩니다:

- "오늘 할일"
- "오늘 뭐해야 해?"
- "할일 정리해줘"
- "투두 알려줘"
- "today"

### 출력 예시

```
📋 오늘의 할일 (2025-01-30 목요일)

📊 어제 할일 현황
- [x] 팀 미팅 참석 ← 캘린더 확인됨 ✅
- [ ] 보고서 작성 ← 오늘로 이월 🔄

🔴 긴급
- [ ] 클라이언트 피드백 반영 (출처: 슬랙 멘션)

🟡 중요  
- [ ] 주간 보고서 마감 (출처: 노션)

🟢 일반
- [ ] 자료 정리 (출처: 노션)

📅 오늘 일정
- 10:00 팀 스탠드업
- 14:00 1:1 미팅

❓ 확인이 필요해요:
"어제 '보고서 작성' 진행하셨나요?"
```

## 🔧 트러블슈팅

| 문제 | 해결 방법 |
|------|----------|
| 슬랙 멘션이 안 잡혀요 | 유저 ID가 정확한지 확인. `to:me`는 동작 안 함 |
| 노션 할일이 안 나와요 | 노션 연결 상태 확인, 태스크 DB에 접근 권한 있는지 확인 |
| 캘린더 일정이 안 보여요 | Google Calendar 연동 상태 확인 |
| 어제 할일이 안 뜨네요 | 최초 사용 시에는 어제 기록이 없어서 정상 |
